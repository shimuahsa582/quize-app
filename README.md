# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, Tailwind CSS, and JavaScript. Users can answer multiple-choice questions and instantly view their scores.

##  Live Demo: https://shimuahsa582.github.io/quize-app/
##  Features

- Multiple-choice quiz questions
- Instant answer validation
- Score calculation
- Responsive design
- Clean and user-friendly interface
- Dynamic question rendering with JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- JavaScript (ES6)

##  How It Works
1. User starts the quiz.
2. Questions are displayed one by one.
3. User selects an answer.
4. The application checks the answer.
5. Final score is displayed after completing the quiz.

##  Responsive Design
The application is optimized for:
- Desktop
- Tablet
- Mobile Devices

##  Future Improvements

- Timer functionality
